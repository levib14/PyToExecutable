# py to exe!
convert your python programs to applications for easy sharing!

# use at web version at:
https://levib14.github.io/PyToExecutable/

# Instructions for web version
input your python program, select the prefered settings, and then click download,
you will find the versions for windows and mac, and then a readme.md file with instructions.

# Python version
you can run this version in a python idle, and it works pretty well, and sometimes better for some people
